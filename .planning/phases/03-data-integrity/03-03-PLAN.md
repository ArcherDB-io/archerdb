Implementation pattern for multiple clients:
```zig
const t = try TestContext.init(.{ .replica_count = 3 });

// Create multiple clients
var c1 = t.clients(.{ .client_count = 1 });
var c2 = t.clients(.{ .client_count = 1 });

// Interleave requests
try c1.request(1, 0);  // Don't wait for this one to commit
try c2.request(1, 0);
t.run();
// StateChecker validates correct interleaving
```
