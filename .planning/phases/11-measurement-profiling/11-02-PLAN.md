<task type="auto">
  <name>Task 1: Create A/B benchmark comparison script</name>
  <files>scripts/benchmark-ab.sh</files>
  <action>
  Create scripts/benchmark-ab.sh that wraps POOP for A/B comparisons:

  1. Accept command line options:
     - `-d, --duration <ms>` - Per-command duration (default: 5000ms)
     - `-w, --warmup <n>` - Warmup iterations (default: 3)
     - `--json` - Output in JSON format for CI
     - `--baseline <cmd>` - Baseline command (first positional arg also works)
     - `--optimized <cmd>` - Optimized command (second positional arg also works)

  2. Verify POOP is installed:
     - Check if `poop` command exists in PATH or tools/poop/zig-out/bin/
     - If not found, print installation instructions:
       ```
       POOP not found. Install with:
         git clone https://github.com/andrewrk/poop tools/poop
         cd tools/poop && zig build -Doptimize=ReleaseFast
       Or set POOP_PATH environment variable.
       ```

  3. Run POOP with provided commands:
     - First command is baseline, second is optimized
     - Use `--duration` flag for sampling duration
     - POOP automatically treats first command as reference

  4. Parse and enhance output:
     - Extract hardware counters (cycles, instructions, cache-refs, cache-misses, branches, branch-misses)
     - Calculate derived metrics (IPC, cache miss rate, branch miss rate)
     - Highlight significant regressions (>5% slower in red)
     - Highlight significant improvements (>5% faster in green)

  5. For JSON output, structure as:
     ```json
     {
       "baseline": { "time_ns": N, "counters": {...} },
       "optimized": { "time_ns": N, "counters": {...} },
       "comparison": {
         "time_delta_percent": N,
         "verdict": "faster|slower|same",
         "significant": true|false
       }
     }
     ```

  Example usage:
  ```bash
  # Compare baseline vs optimized binary
  ./scripts/benchmark-ab.sh \
    './archerdb-baseline benchmark --quick' \
    './archerdb-optimized benchmark --quick'

  # With explicit flags
  ./scripts/benchmark-ab.sh -d 10000 \
    --baseline './old benchmark' \
    --optimized './new benchmark'
  ```
  </action>
  <verify>
  ```bash

  chmod +x scripts/benchmark-ab.sh
  scripts/benchmark-ab.sh --help
  ```
  </verify>

     - Compare: `./scripts/benchmark-ab.sh './baseline benchmark' './zig-out/bin/archerdb benchmark'`
     - Iterate until improvement is statistically significant
