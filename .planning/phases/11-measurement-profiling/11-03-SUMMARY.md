## Decisions Made

1. **Simple allocator wrapper over DebugAllocator embedding** - The initial approach tried to embed std.heap.DebugAllocator directly, but Zig 0.14.1 API uses constant initialization. Changed to a simpler wrapper that accepts any allocator, with separate DebugTrackingAllocator for stack trace support.

2. **ExtendedStats as public struct** - Placed ExtendedStats outside HistogramType function to make it easily accessible for type annotations and independent use.

3. **Prometheus quantile format** - formatExtended() outputs quantile labels compatible with Prometheus summary format for integration with existing monitoring.
