<task type="auto">
  <name>Task 2: Create compression module with block-level functions</name>
  <files>src/lsm/compression.zig</files>
  <action>
  Create src/lsm/compression.zig with block-level compression primitives:

  1. Define CompressionType enum:
     ```zig
     pub const CompressionType = enum(u4) {
         none = 0,
         lz4 = 1,
         // Reserved: zstd = 2 (future expansion)
     };
     ```

  2. Import LZ4 via @cImport:
     ```zig
     const lz4 = @cImport({
         @cInclude("lz4.h");
     });
     ```

  3. Implement compress_block():
     - Takes input slice and output buffer
     - Returns struct { size: usize, compression_type: CompressionType }
     - Only use compression if it saves at least 10% space (90% threshold)
     - Fall back to none (memcpy) if compression doesn't help
     - Handle LZ4_compress_default return values correctly

  4. Implement decompress_block():
     - Takes compressed input, output buffer, compression type, and original size
     - For CompressionType.none: memcpy
     - For CompressionType.lz4: LZ4_decompress_safe
     - Return error.DecompressionFailed on LZ4 errors

  5. Implement helper functions:
     - max_compressed_size(input_len): Return LZ4_compressBound
     - is_compressible(compression_type): Return true if not .none

  6. Add unit tests:
     - Test round-trip: compress then decompress equals original
     - Test incompressible data falls back to none
     - Test various block sizes (small, typical 64KiB, edge cases)

  Follow existing Zig patterns in src/lsm/*.zig for error handling and assertions.
  </action>
  <verify>
