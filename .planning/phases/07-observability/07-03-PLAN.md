  <action>
Create new dashboard `observability/grafana/dashboards/archerdb-unified-overview.json` following CONTEXT.md requirements.

Key structure:
1. **Dashboard metadata:**
   - uid: "archerdb-unified-overview"
   - title: "ArcherDB Unified Overview"
   - Default time range: "now-1h" to "now"
   - Refresh: 30s
   - Variables: datasource, instance (multi-select)

2. **Row 1: Cluster Status** (green/yellow/red indicators)
   - Stat panel per node showing health status
   - Thresholds: green (1 = healthy), yellow (0.5 = degraded), red (0 = down)
   - Query: `archerdb_health_status{status="ready", instance=~"$instance"}`
   - Use valueMapping: 1=Healthy (green), 0=Down (red), else=Degraded (yellow)

3. **Row 2: Throughput & Latency** (dual Y-axis panel)
   - Left Y-axis: Throughput (ops/sec) as bars
   - Right Y-axis: P99 latency (ms) as lines
   - Queries:
     - Throughput: `sum(rate(archerdb_read_operations_total{instance=~"$instance"}[5m])) + sum(rate(archerdb_write_operations_total{instance=~"$instance"}[5m]))`
     - Read P99: `histogram_quantile(0.99, sum(rate(archerdb_read_latency_seconds_bucket{instance=~"$instance"}[5m])) by (le))`
     - Write P99: `histogram_quantile(0.99, sum(rate(archerdb_write_latency_seconds_bucket{instance=~"$instance"}[5m])) by (le))`
   - Override: latency series use right Y-axis, unit: seconds

4. **Row 3: Resource Usage**
   - CPU usage: `rate(process_cpu_seconds_total{job=~"archerdb.*", instance=~"$instance"}[5m])`
   - Memory usage: `process_resident_memory_bytes{job=~"archerdb.*", instance=~"$instance"}`
   - Disk usage: `1 - (node_filesystem_avail_bytes{mountpoint=~"/data|/var/lib/archerdb"} / node_filesystem_size_bytes{...})`

5. **Row 4: Key Internals**
   - Replication lag (if multi-node): `archerdb_replication_lag_seconds`
   - Cache hit ratio: `archerdb_cache_hit_ratio{cache_type="s2_covering"}`
   - Compaction pending: `archerdb_compaction_pending_bytes`

Use existing archerdb-overview.json as structural template for panel sizing, colors, and variable definitions.

Generate valid Grafana 9.x JSON schema (schemaVersion: 39).
