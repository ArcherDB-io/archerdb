- `generate_report(results: ScalabilityResult) -> str` - Human-readable report

ScalabilityResult dataclass:
- topologies_tested: List[int]
- throughput_by_topology: Dict[int, float]  # events/sec
- scaling_factors: Dict[int, float]  # efficiency vs linear
- is_linear: bool  # True if all factors 0.8-1.2
- timestamp: str
