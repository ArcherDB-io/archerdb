Implement load shedding to reject requests under overload before cascading failure occurs.

Purpose: Under overload, accepting all requests leads to cascading failure (increased latency, memory exhaustion, timeout storms). Load shedding protects the system by rejecting excess requests early with clear feedback (429 + Retry-After), allowing the system to maintain quality of service for accepted requests.

Output: `src/load_shedding.zig` with LoadShedder struct, cluster_metrics.zig extended with shedding metrics.

Create load shedding implementation following CONTEXT.md decisions.

Per user decisions:
- Primary overload signal: Composite (queue depth + latency P99 + resource pressure)
- Shedding curve: Hard cutoff (below threshold accept all, above reject all)
- Shed response: 429 Too Many Requests with Retry-After header
- Runtime configurability: Yes, but with guardrails (cannot disable entirely)

Key structs:
```zig
pub const ShedConfig = struct {
    // Composite signal weights (default equal weighting)
    queue_depth_weight: f32 = 0.34,
    latency_p99_weight: f32 = 0.33,
    resource_pressure_weight: f32 = 0.33,

    // Hard cutoff threshold (guardrails: 0.5-0.95)
    threshold: f32 = 0.8,
    min_threshold: f32 = 0.5,   // Cannot set lower
    max_threshold: f32 = 0.95,  // Cannot set higher

    // Signal thresholds for normalization
    max_queue_depth: u32 = 10000,
    max_latency_p99_ms: u64 = 1000,
    max_memory_pressure_pct: u8 = 90,

    // Retry-After calculation
    base_retry_ms: u64 = 1000,
    max_retry_ms: u64 = 30000,
};

pub const ShedDecision = struct {
    shed: bool,
    retry_after_ms: ?u64,
    score: f32,  // Current composite score for metrics
    reason: ?ShedReason,
};

pub const ShedReason = enum {
    queue_depth,
    latency_p99,
    resource_pressure,
    composite,
};

pub const LoadShedder = struct {
    config: ShedConfig,
    metrics: *ClusterMetrics,

    // Current signal values
    queue_depth: std.atomic.Value(u32),
    latency_p99_ns: std.atomic.Value(u64),
    memory_used_pct: std.atomic.Value(u8),

    pub fn shouldShed(self: *Self) ShedDecision { ... }
    pub fn updateQueueDepth(self: *Self, depth: u32) void { ... }
    pub fn updateLatencyP99(self: *Self, latency_ns: u64) void { ... }
    pub fn updateMemoryPressure(self: *Self, used_pct: u8) void { ... }

    fn computeCompositeScore(self: *Self) f32 { ... }
    fn computeRetryAfter(self: *Self, score: f32) u64 { ... }
};
```

Composite score calculation:
- Normalize each signal to 0.0-1.0 range
- queue_score = min(1.0, queue_depth / max_queue_depth)
- latency_score = min(1.0, latency_p99_ms / max_latency_p99_ms)
- memory_score = min(1.0, memory_used_pct / max_memory_pressure_pct)
- composite = queue_weight * queue_score + latency_weight * latency_score + resource_weight * memory_score

Retry-After calculation: Exponential backoff based on how far over threshold. `base_retry_ms * (1 + (score - threshold) * 10)`, capped at max_retry_ms.
