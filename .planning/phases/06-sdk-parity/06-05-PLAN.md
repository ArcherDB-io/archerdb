## Async Client

For asyncio applications, use `GeoClientAsync`:

```python
import asyncio
from archerdb import GeoClientAsync, GeoClientConfig, GeoEvent

async def main():
    config = GeoClientConfig(
        cluster_id=0,
        addresses=["127.0.0.1:3001"],
    )

    async with GeoClientAsync(config) as client:
        # Insert events
        batch = client.create_batch()
        batch.add(GeoEvent(
            entity_id=123,
            latitude=37.7749,
            longitude=-122.4194,
        ))
        errors = await batch.commit()

        # Query events
        result = await client.query_radius(
            latitude=37.7749,
            longitude=-122.4194,
            radius_m=1000,
        )

        for event in result.events:
            print(f"Found: {event.entity_id}")

asyncio.run(main())
```

```markdown
## Error Handling

ArcherDB uses typed exceptions:

```python
from archerdb import (
    ArcherDBError,
    InvalidCoordinates,
    ConnectionFailed,
    OperationTimeout,
    RetryExhausted,
)

try:
    result = client.query_radius(latitude=37.7749, longitude=-122.4194, radius_m=1000)
except InvalidCoordinates as e:
    # Invalid input - don't retry
    print(f"Invalid coordinates: {e}")
except ConnectionFailed as e:
    # Network error - retryable
    print(f"Connection failed (retryable={e.retryable}): {e}")
except OperationTimeout as e:
    # Timeout - may have committed
    print(f"Operation timed out: {e}")
except RetryExhausted as e:
    # All retries failed
    print(f"Retries exhausted: {e.last_error}")
except ArcherDBError as e:
    # Catch-all for other errors
    print(f"Error {e.code}: {e}")
```

### Error Categories

| Exception | Codes | Retryable | Description |
|-----------|-------|-----------|-------------|
| ConnectionFailed | 1001 | Yes | Network connectivity issues |
| ConnectionTimeout | 1002 | Yes | Connection attempt timed out |
| ClusterUnavailable | 2001 | Yes | Cluster not available |
| InvalidCoordinates | 3001 | No | Coordinates out of range |
| BatchTooLarge | 3003 | No | Batch exceeds maximum size |
| OperationTimeout | 4001 | Yes | Operation timed out |
| RetryExhausted | 5002 | No | All retry attempts failed |
```
